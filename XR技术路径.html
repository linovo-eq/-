Birdbath（鸟浴盆）
传统半反半透光学方案
原理介绍
通过半反半透镜反射微型显示屏（Micro OLED）图像到人眼，同时让外部光线通过。结构简单，亮度高。
制作流程
  1. 1Micro OLED制造
  2. 2光学镜片切割与抛光
  3. 3分光镜镀膜
  4. 4模组组装与调校
  5. 5整机测试
视场角广、亮度高、成本低、成熟工艺
体积较大、重影、透光率有限

光波导 - SRG表面浮雕光栅
主流消费级AR光波导方案之一
原理介绍
通过微纳光栅将Micro OLED/Micro LED图像耦合进玻璃波导，内部全反射传输，再用出射光栅投射到人眼。可做极薄镜片。
制作流程
  1. 1光栅结构设计
  2. 2纳米压印/光刻
  3. 3刻蚀、薄膜沉积
  4. 4多层RGB组装
  5. 5整机组装调试
镜片极薄、可批量生产、外观接近普通眼镜
光效低、色散明显、彩虹纹问题

光波导 - VHG/PVG体全息光栅
高衍射效率单色/双色光波导方案
原理介绍
利用体积全息材料实现高效衍射，单色波长衍射效率极高，光损低。多色需多层堆叠。
制作流程
  1. 1全息感光材料涂布
  2. 2激光曝光生成体全息光栅
  3. 3化学处理定型
  4. 4多层堆叠组装（多色）
衍射效率极高，光损耗小
目前主流为单色，三色厚度大，未大规模商用

光波导 - 几何波导
微型反射镜阵列，专业级AR主流方案，自由曲面为其子类
原理介绍
利用波导片中微型反射镜阵列进行光线分束/耦合/均匀输出。无色散，耦合效率高，像差校正优异。
制作流程
  1. 1高精密微结构设计
  2. 2超精密切割、车削
  3. 3多层镀膜
  4. 4阵列组装、调试
高耦合效率、无色散、像差控制好
制造难度极高，量产良率低
代表产品

自由曲面（几何波导子类）
通过特殊复杂曲面形状控制像差，属于专业级几何波导方案
原理介绍
采用自由曲面（非旋转对称曲线），对光线逐点精准修正，实现极佳成像质量。结构薄、轻，且FoV大。
制作流程

  2. 2表面精密抛光
  3. 3多层镀膜
  4. 4组装校准
像差修正优异、结构极薄、可实现大视场
工艺难度高，批量良率低

Pancake光学方案
目前消费级VR头显主流选择
原理介绍
利用多片偏振反射镜和折叠光路设计，将光线反射多次后成像于人眼，大幅缩短光路，实现更薄的VR头显。
制作流程
  1. 1Micro OLED / LCD面板制造
  2. 2偏振膜组装与镀膜
  3. 3Pancake光学模组组装
  4. 4整机调校与测试
体积小、重量轻、支持屈光度调节、佩戴舒适
光效损耗大、对偏振元件要求高、成本较高

超表面光学（Metasurface）
亚波长纳米结构引领未来显示
原理介绍
利用亚波长的人工纳米结构（超表面）调控光传播，实现光的折射、反射、色散和聚焦能力的突破，有望实现极致轻薄和无色散的AR/VR光学。
制作流程
  1. 1半导体材料沉积（如氮化硅/氧化钛）
  2. 2电子束/纳米压印微纳图形制作
  3. 3反应离子刻蚀成型
  4. 4与显示模组集成
理论可消除色散、极致轻薄、光路可编程
量产工艺难、损耗高、商用初期
